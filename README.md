# 🔐 Ultimate Password Generator

A modern and secure password generator built with **HTML, CSS, JavaScript, and Bootstrap**. Generate strong, customizable passwords instantly with a clean glassmorphism interface and useful security features.

---

## ✨ Features

* 🔒 Cryptographically secure password generation
* 🎚️ Adjustable password length
* 🔤 Include uppercase and lowercase letters
* 🔢 Include numbers
* 🔣 Include special characters
* 🚫 Option to prevent repeating characters
* 📊 Real-time password strength meter
* 📈 Password entropy calculation
* 👁️ Show/Hide password
* 📋 One-click copy to clipboard
* 💾 Save password as a TXT file
* 🕒 Password history using Local Storage
* 📱 Fully responsive design
* 🎨 Modern glassmorphism user interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
* Bootstrap Icons

---

## 🚀 Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/ultimate-password-generator.git
   ```

2. Open the project folder.

3. Run `index.html` in any modern web browser.

No installation or additional dependencies are required.

## 🔐 Security Features

* Uses the Web Crypto API (`crypto.getRandomValues`) for secure random password generation.
* Calculates password strength based on length and character diversity.
* Displays estimated password entropy.
* Stores password history only in the browser using Local Storage.
